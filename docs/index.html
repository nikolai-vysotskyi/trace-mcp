001 / Premise

AI agents are
language-aware but framework-blind.

Without trace-mcp

  • Agent reads 15+ files to understand one feature
  • Doesn't know which Vue page a controller renders
  • "What breaks if I change this?" — agent guesses
  • Schema requires a running database
  • Prop mismatch between PHP and Vue surfaces in production
  • Burns 600K+ tokens per exploration session

With trace-mcp

  • get_task_context — optimal subgraph in one shot
  • routes_to → renders_component → uses_prop edges
  • get_change_impact traverses reverse deps across languages
  • Migrations parsed — schema reconstructed from code
  • Prop mismatch detected at index time
  • 94% fewer tokens — ~624K saved per session
002 / Capabilities

Six things no other tool does.

/ 01

Framework-aware edges

Understands that Inertia::render('Users/Show') connects PHP to Vue. 58 integrations across 15 frameworks, 7 ORMs, 13 UI libraries.

/ 02

Code-linked decision memory

"Chose PostgreSQL for JSONB" is linked to Pool#class. When someone runs get_change_impact, the decision surfaces automatically.

/ 03

Cross-session intelligence

Past sessions are mined for decisions and indexed for search. get_wake_up gives orientation in ~300 tokens.

/ 04

Request flow tracing

URL → Route → Middleware → Controller → Service → View, across 15 backend frameworks. One call.

/ 05

Security scanning

OWASP Top-10 pattern scanning and taint analysis — traces untrusted data from source to sink across function boundaries.

/ 06

Automated refactoring

Rename across files, extract functions, move code, apply codemods — all graph-aware, all reversible, with diff preview.

005 / Install

Three commands.
No configuration.

01

/ Install Globally

npm install -g trace-mcp

Single binary. No native dependencies. Runs on Node 20+.

02

/ Initialize Once

trace-mcp init

Configures your MCP client (Claude Code, Cursor, Windsurf), installs the guard hook, updates CLAUDE.md.

03

/ Add A Project

trace-mcp add

Detects frameworks, indexes the codebase, registers the project. Run in each repo you want indexed.

~/projects/my-app session 01
# install + initialize $ npm install -g trace-mcp $ trace-mcp init [OK] claude code configured [OK] guard hook installed [OK] claude.md updated # register project $ cd ~/my-app $ trace-mcp add [OK] detected: laravel, vue, inertia [OK] indexed 1,247 files [OK] 8,392 symbols / 14,902 edges # use in your AI agent > get_task_context("fix login bug") [READY]
